# openclaw-kali-lab

A hands-on security learning journal — built in public by a practitioner, not a developer.

Trevor is a finance and governance professional learning Linux, networking, and security tooling from the ground up. This repo documents the journey: the lab setup, the techniques learned, the mistakes made, and the progress over time.

Guided by **Scout 🔍** — an OpenClaw agent running on Kali Linux.

---

## Philosophy

Learn by doing. Document everything. Understand the why, not just the how.

Strictly ethical: all techniques practiced on owned lab equipment only.

---

## Structure

| Path | What it is |
|------|------------|
| `environment.md` | Lab setup — hardware, VMs, network layout |
| `journal/` | Dated session notes |
| `targets/` | Notes on lab target machines |
| `skills/` | Concepts and tools — documented as learned |

---

## Lab Environment

- **Attacker:** Kali Linux (VMware VM, Windows 11 host)
- **Agent:** Scout 🔍 (OpenClaw, Telegram)
- **Network:** VMware NAT — 192.168.238.0/24
